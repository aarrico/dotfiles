Window & Pane Mechanics Tmux (Prefix: Ctrl+a)

C-a then | Splits active pane Horizontally at the exact current path link.
C-a then - Splits active pane Vertically at the exact current path link.
C-a then c Creates a fresh Window path layer context.
C-a then h/j/k/l Directional vim navigation between splits (Left, Down, Up, Right).
